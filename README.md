demo_go_bootstrap
=================

Demo :: Go with bootstrap


For MAC

```
$ git clone git://github.com/up1/demo_go_bootstrap.git
$ cd demo_go_bootstrap
$ ./build.sh
$ ./bin/server
```

For Windows

```
> git clone git://github.com/up1/demo_go_bootstrap.git
> cd demo_go_bootstrap
> build.bat
> bin\server.exe
```


Run

* http://localhost:8080
* http://localhost:8080/xxxx
